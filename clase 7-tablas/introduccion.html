TABLAS

¿que es una tabla?

Una tabla es un conjunto estructurado de datos distribuidos en filas y columnas (datos tabulados). Una tabla permite buscar con rapidez y facilidad valores entre diferentes tipos de datos que indiquen algún tipo de conexión. Por ejemplo, una persona y su edad, o un día de la semana o el horario de una piscina municipal.

¿Qué etiquetas conforman una tabla en html?

  • <table>:

    Esta etiqueta define la tabla en sí misma y envuelve todo el contenido de la tabla

  • <tr>:

    Esta etiqueta define una fila de la tabla.

  • <th>:

    Esta etiqueta define una celda de encabezado en una fila de la tabla. Las celdas de encabezado se utilizan comúnmente para mostrar los títulos de las columnas.

  • <th>:

    Esta etiqueta define una celda de encabezado en una fila de la tabla. Las celdas de encabezado se utilizan comúnmente para mostrar los títulos de las columnas.

  • <td>:

    Esta etiqueta define una celda de datos en una fila de la tabla. Las celdas de datos se utilizan para mostrar los datos de la tabla.

  • <thead>:

    Esta etiqueta define el encabezado de la tabla y envuelve las filas de encabezado .

  • <tbody>:

    Esta etiqueta define el cuerpo de la tabla y envuelve las filas de datos .

  • <tfoot>:

    Esta etiqueta define el pie de la tabla y envuelve las filas de resumen .

  • <caption>:

    Esta etiqueta se utiliza para agregar un título o descripción a la tabla .

Abarcar varias celdas o columnas

Para hacer que un dato abarque varias celdas o columnas en una tabla HTML, se utiliza la etiqueta td o th con el atributo colspan o rowspan.

El atributo colspan se utiliza para especificar el número de columnas que el dato debe abarcar. Por ejemplo, si se quiere que un dato ocupe dos columnas, se utiliza colspan="2"

El atributo rowspan se utiliza para especificar el número de filas que el dato debe abarcar. Por ejemplo, si se quiere que un dato ocupe dos filas, se utiliza rowspan="2"
