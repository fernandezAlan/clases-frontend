PROYECTOS

#1

Formulario de inicio de sesion

Este proyecto es un formulario de inicio de sesión diseñado con HTML y CSS. Incluye campos para ingresar correo electrónico y contraseña, botones para iniciar sesión o registrarse, y una opción de "Recordarme". El diseño es moderno y minimalista, con una combinación suave y elegante de colores.

#2

portfolio de fotografo

Este proyecto es un portafolio web diseñado para un fotógrafo profesional. Incluye una página de inicio con una galería de imágenes y una sección de portafolio organizada en categorías. El diseño es minimalista y limpio, con una paleta de colores suave que resalta el trabajo del fotógrafo. También cuenta con una sección de contacto para que los visitantes puedan enviar un correo electrónico al fotógrafo.
